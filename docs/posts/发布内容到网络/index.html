使用Github Pages服务发布内容到真实互联网

December 11, 2023

注册Github账号

参见Github官方文档

新建Github仓库

要点：一定要将仓库设置为公开（public）

将Github仓库克隆到本地

  1. 找到Github仓库地址，复制，然后在本地计算机的桌面上按右键，选择git bash here，启动git bash
  2. 执行git clone xxxxxxxx就是前面复制的Github仓库地址

合并之前创建的hugo项目和Github仓库

  1. 将之前创建的hugo项目中的所有文件和文件夹（除.git文件夾）复制到本地Github仓库中。

开启Github Pages服务

  1. 进入Github仓库的设置页面，选择pages，将来源设置为仓库，分支设置为main，目录设置为docs。
  2. 然后选择save，保存并启动设置。

配置合并后的hugo项目

打开hugo项目配置文件，如hugo.toml,增加两项设置：

baseURL = 'https://yangjh-xbmu.github.io/hugonews/'
languageCode = 'zh'
publishDir = "docs"
theme = 'ananke'
title = 'My New Hugo Site'

新建发布脚本

在hugo项目的根目录中，创建名为push.sh的文件，内容如下：

hugo
git add .
git commit -m "自动推送"
git push

使用脚本发布本地仓库内容到Github远程仓库

在VS Code中，其中bash终端，执行如下命令：

sh push.sh

该脚本将执行hugo建构程序，并将内容推送到Github仓库，之后Github会自行调用Actions，发布仓库内容中的docs目录到pages服务。
